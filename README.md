# WanderLust

A full-stack Airbnb-inspired web application that allows users to discover, create, edit, and manage property listings. Users can sign up, log in securely, leave reviews, and explore listings through an interactive and responsive interface.

## Features

* User authentication and authorization
* Create, edit, and delete property listings
* Add and manage reviews and ratings
* Secure session management
* Interactive map integration for property locations
* Flash messages and error handling
* Responsive user interface using EJS templates
* MongoDB database integration with Mongoose

## Tech Stack

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose

### Frontend

* EJS
* HTML5
* CSS3
* JavaScript

### Authentication & Security

* Passport.js
* Express Session
* Connect Flash

### Other Tools

* Cloudinary (Image Storage)
* Map Integration
* Joi Validation

## Project Goal

This project replicates the core functionality of Airbnb while demonstrating full-stack web development concepts such as RESTful routing, authentication, database management, middleware, MVC architecture, and deployment-ready application design.
